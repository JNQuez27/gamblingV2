# Color Palette

This document outlines the color palette used in the application.

| Color Name | Hex Code | Usage |
| :--- | :--- | :--- |
| Background | `#020817` | Main background color |
| Foreground | `#f8fafc` | Main text color |
| Card | `#0f172a` | Card background color |
| Card Foreground | `#f8fafc` | Card text color |
| Popover | `#020817` | Popover background color |
| Popover Foreground | `#f8fafc` | Popover text color |
| Primary | `#f97316` | Primary action color (buttons, links) |
| Primary Foreground | `#f8fafc` | Text on primary elements |
| Secondary | `#334155` | Secondary elements |
| Secondary Foreground | `#f8fafc` | Text on secondary elements |
| Muted | `#334155` | Muted elements |
| Muted Foreground | `#94a3b8` | Muted text color |
| Accent | `#475569` | Accent color |
| Accent Foreground | `#f8fafc` | Text on accent elements |
| Destructive | `#ef4444` | Destructive actions (e.g., delete) |
| Destructive Foreground | `#f8fafc` | Text on destructive elements |
| Border | `#1e293b` | Border color |
| Input | `#1e293b` | Input field background |
| Ring | `#f97316` | Focus ring color |

## Category Gradient Colors

| Category | Gradient |

| Sleep | `from-indigo-400` to `to-purple-600` |
| Stress | `from-orange-400` to `to-red-500` |
| Focus | `from-green-400` to `to-emerald-600` |
| Body Scan | `from-pink-400` to `to-rose-600` |
| Loving Kindness | `from-red-400` to `to-rose-500` |
